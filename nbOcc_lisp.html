nbOcc version lisp

description

chercher le nombre d'occurence x dans la liste l

code nbOcc

  (defun nbOcc(l x)
    (if (endp l)
    0
    (+ (if (equal(car l)x)
         1 0) (nbOcc(cdr l) x))))
